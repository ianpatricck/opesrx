# OpesRX

OpesRX é um aplicativo **de gestão financeira pessoal** desenvolvido em **React Native com Expo**, projetado para ajudar usuários a **controlar seus ganhos e gastos** de maneira simples e visual. O aplicativo permite adicionar transações, visualizar resumos financeiros, consultar o histórico e acompanhar gráficos mensais de entradas e saídas.

---

## Funcionalidades

- Adicionar, visualizar e remover transações (ganhos e gastos).
- Tela de **Resumo Financeiro** com saldo, entradas e saídas.
- Tela de **Histórico de Transações** com lista detalhada.
- Gráfico **Mensal de Ganhos e Gastos**, mostrando barras lado a lado de entradas e saídas.
- Atualização automática do gráfico ao adicionar ou remover transações.

---

## Tecnologias Utilizadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [SQLite](https://www.sqlite.org/index.html) (armazenamento local de dados)
- [React Navigation](https://reactnavigation.org/) (navegação entre telas)
- [react-native-gifted-charts](https://github.com/indiespirit/react-native-gifted-charts) (gráficos interativos)
- TypeScript

## Instalação e Execução Local

### Pré-requisitos

- Node.js >= 18.x
- npm ou yarn
- Expo Go no dispositivo móvel (Android ou iOS)

### Passos

1. **Clonar o repositório**

```bash
$ git clone https://github.com/seu-usuario/opesrx.git
$ cd opesrx
```

2. **Instalar as dependencias e rodar a aplicação**

```bash
$ npm install
$ npx expo start
```
